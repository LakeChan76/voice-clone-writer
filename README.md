# Voice Clone Writer — IP风格润色师

> 让AI改完的内容，还像你自己写的。

## 这是什么？

一个为TRAE（或其他支持SKILL.md的AI编辑器）设计的内容润色Skill。

**核心能力：先学习你的个人语言风格，再用你的风格去改写内容**，而不是用一套"通用好文笔"去覆盖所有人。

你不生产标准答案，你生产"这个人会说的话"。

## 解决什么问题？

用AI帮你改文章，改完发现——不像自己写的了：
- 句子太顺、太干净、太"正确"
- 你自己的口头禅没了，你的节奏没了
- 换了个AI工具再改一遍，还是一样——因为所有AI都在往"通用好文笔"方向改

这个Skill的解法是：**先让AI学会你是谁，再让它用你的风格改**。

## 怎么用？

### 第一步：安装

1. 把本仓库下载/克隆到本地
2. 将 `.trae/skills/voice-clone-writer/` 整个目录复制到你的TRAE工作目录下
3. 在TRAE中输入 `Use Skill: voice-clone-writer` 即可调用

### 第二步：建立你的风格档案（只需一次）

第一次使用会自动引导你完成风格采集，提供三种方式：

| 方式 | 说明 | 耗时 |
|---|---|---|
| **A. 10道风格选择题** | 覆盖语气、句式、情绪、幽默等10个维度 | 2分钟 |
| **B. 上传过往文章** | 贴你写过的内容，AI自动分析高频词、句式、开头收尾习惯 | 取决于文章长度 |
| **C. 自由问答** | 像聊天一样回答3-5个问题，AI从中提取你的表达习惯 | 3-5分钟 |

采集完成后，会自动生成 `voice-profile.md` 风格档案，后续直接调用，无需重复采集。

### 第三步：润色内容

输入你想润色的小红书正文（500字以内），Skill会按以下四步执行：

1. **违规词检查**：扫描粗口、敏感词、平台高风险词
2. **AI味检测**：17项指纹扫描（排比、翻译腔、情绪太光滑……）
3. **IP风格改写**：用你的档案把AI味替换成你的表达方式
4. **对比输出**：原文 vs IP版 逐句对比，每处改写标注理由

## 效果示例

**润色前：**
> 我愣了三秒，现在一想当时回答的确实不太行。

**润色后：**
> 我当场卡壳了，说了个啥我现在都不好意思回忆。

**改写理由：** 档案显示博主习惯用"卡壳"而非"愣住"，"不好意思回忆"比"不太行"更有态度。

## 适合谁用？

- 有自己写作风格但被AI改没了的博主
- 想要建立个人IP辨识度，不想内容千篇一律
- 需要批量产出但每篇都要保持"自己味道"的内容运营

## 注意事项

- 风格档案只需建立一次，后续直接调用
- 目前只针对**小红书图文正文（500字以内）**优化
- 档案是你的个人数据，不要上传到公开仓库（已默认加入 `.gitignore`）
- 不支持公众号长文、朋友圈文案、短视频脚本

## 核心原则

1. **风格优先于正确**：改完读起来像你写的，哪怕有口语化、有毛边
2. **改的不是文字，是态度**：每处修改都能追溯到"你本人会怎么表达"
3. **有档案就用档案，没档案就先建**：一次采集，终身使用
